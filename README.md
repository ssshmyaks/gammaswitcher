# Gamma Switcher

Toggle screen gamma/brightness with a customizable hotkey.

## Features
- Global hotkey (default F12)
- Customizable hotkey via tray menu
- Runs in system tray
- Settings persist after restart

## Usage
1. Run `gammaswitcher.exe`
2. Press F12 to toggle gamma
3. Right-click tray icon → "Change Key" to customize

## Build
dotnet build -c Release


## Requirements
- Windows 10/11
- .NET 8.0 or higher
